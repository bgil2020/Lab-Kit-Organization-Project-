# Lab-Kit-Organization-Project-
This project is designed to improve the current engineering lab kit assembly and distribution at FAU.
Currently the kits are assembled manually and students have to go through and ensure thry have all of the required components for the course. This involves a lot of confusion and often misidentiifcation due to inexperience. 
This project aims to fix the problem by having each kit be assosiated with a QR Code.
